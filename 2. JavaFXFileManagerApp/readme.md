# Disc Manager

## General information
File manager

## Description:
Simple files manager wit Gui created with JavaFX.

### Features

## Technologies
- Java 8
- JavaFX

## Setup
Requirements:
- Ant 1.8.0 or higher
- Java 8 or higher
To build this project locally open project directory and run:
```
ant compile 
ant jar
```
To run program from Command Prompt run follow: 
```
java -jar dist\JavaFXFileManagerApp.jar
```
